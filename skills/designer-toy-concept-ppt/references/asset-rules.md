# Asset Rules

## Default

Generate original mascots and graphics with imagegen. Do not imitate specific commercial characters.

## Optional Open Asset Sources

If using external illustration assets, save source and license notes in `research/material-index.md` or `research/license-notes.md`.

Suggested source classes:

- Open Peeps: friendly hand-drawn people.
- OpenMoji: open emoji/icon system; check attribution/share-alike.
- Twemoji: emoji graphics; check attribution.
- Blush: illustration library; check selected asset license.
- unDraw: flat illustrations; check license.

## User-Provided Commercial IP

If the user provides an image and says they have the right to use it:

- use it only as placed/reference material
- keep source and authorization note in the workspace
- do not generate a new imitation or derivative copycat character
