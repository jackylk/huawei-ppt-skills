# Designer Toy Concept Style Guide

## Goal

Create a cute, original designer-toy concept deck that is friendly and memorable while still carrying a strategic argument.

## Visual Language

- Original soft-vinyl mascot characters.
- Blind-box cards, toy shelves, capsule labels, sticker tags, soft shadows.
- Pastel palette: cream, mint, sky blue, lavender, coral pink, butter yellow.
- Dark slate text for readability.
- Rounded rectangles and simple iconography.
- Low-to-medium density: one claim, one main visual, 2-4 supporting blocks.

## Original Mascot Rules

Characters may be:

- round-headed
- soft vinyl or clay-like
- expressive and friendly
- wearing simple original accessories that represent roles
- small enough not to overwhelm the slide

Characters must not:

- resemble or reference Labubu, Molly, Pop Mart characters, Disney, Pokemon, Doraemon, or other specific commercial IP
- copy distinctive ears, teeth, eyes, costumes, packaging, names, or silhouettes
- use brand logos or fake official merchandise

## Good Page Types

- Toy-lab cover
- Blind-box agenda
- Character scenario storyboard
- Capability shelf
- Friendly architecture map
- Roadmap islands
- Role cards
- Before/after journey

## Technical Content Rule

Even in cute visual style, keep real labels for technical concepts. For example:

- Good: `Context Graph`, `Policy Gate`, `Tool Runtime`, `Evaluation Loop`
- Weak: `magic brain`, `smart helper`, `cute robot`

Use cute visuals to make the page approachable, not vague.
